# Kana-Kona 💬

A real-time chat application built using the MERN stack with WebSocket communication.

## 🚀 Live Demo
  https://kana-kona.vercel.app

---

## 🧰 Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Socket.IO
* JWT Authentication
* Cloudinary (image uploads)

### Database

* MongoDB Atlas

### Deployment

* Vercel (Frontend)
* Render (Backend)

---

## ✨ Features

* User authentication (JWT)
* Real-time messaging using Socket.IO
* Online user status
* Typing indicator
* Image sharing
* Secure cookies
* Cloud image storage

## 🛠️ Installation

Clone the repository

```
git clone https://github.com/Yadhu-Krishnan-K/Kana-Kona.git
```

Backend setup

```
cd Backend
npm install
npm run dev
```

Frontend setup

```
cd Frontend
npm install
npm run dev
```
